# Сайт Акима

Готовый статический сайт для публикации через GitHub Pages.

## Что лежит в проекте

- `index.html`
- `style.css`
- `script.js`
- `akim-1.jpg`
- `akim-2.jpg`
- `.nojekyll`

## Как залить на GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи в него все файлы из этой папки.
3. Убедись, что `index.html` лежит в корне репозитория.

## Как включить GitHub Pages

1. Открой репозиторий на GitHub.
2. Зайди в `Settings`.
3. Открой раздел `Pages`.
4. В блоке `Build and deployment` выбери:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Нажми `Save`.

Через минуту GitHub Pages даст публичную ссылку на сайт.

## Важно

- Файлы `akim-1.jpg` и `akim-2.jpg` должны оставаться рядом с `index.html`.
- Если меняешь сайт, просто загрузи новые версии файлов в репозиторий.
- После обновления GitHub Pages автоматически обновит сайт.
